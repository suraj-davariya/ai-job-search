تأخذك هذه الصفحة من جهاز فارغ إلى أول سيرة ذاتية وخطاب تغطية مخصّصين لك. خطّط
لنحو ثلاثين دقيقة في المرة الأولى؛ معظمها هو تثبيت LaTeX يقوم بعمله بهدوء.

## ما ستحتاجه أولًا

كن صادقًا مع نفسك بشأن هذه المتطلبات الثلاثة — كلٌّ منها يرتبط بدليل تثبيت
مناسب، ولا يتطلّب أيّ منها برمجة:

1. **Claude Code** — المساعد الذكي الذي يعمل CareerForge داخله. إنه برنامج
   تثبّته مرة واحدة وتفتحه من نافذة طرفية (terminal). لم تستخدم الطرفية من قبل؟
   لا بأس: لن تكتب سوى أوامر قصيرة معروضة في هذه الصفحة.
2. **LaTeX** — نظام التنضيد الذي يحوّل سيرتك الذاتية إلى ملف PDF مصقول. إنه
   تنزيل كبير، لكنه يُثبَّت ويُنسى.
3. **Python 3.10+** *(اختياري)* — يلزم لاحقًا فقط لمقارنة الرواتب المرجعية.
   تخطَّه اليوم.

<Steps>

<Step>

### ثبّت Claude Code

```bash title="Terminal"
npm install -g @anthropic-ai/claude-code
```

ثم شغّله مرة واحدة لتسجيل الدخول:

```bash title="Terminal"
claude
```

ستحتاج إلى [حساب Anthropic](https://claude.ai) — تعمل خطتا Free وPro كلتاهما.

**ما يحدث بعد ذلك:** تُفتح محادثة في الطرفية. نافذة المحادثة تلك هي حيث
ستشغّل كل ما يلي.

</Step>

<Step>

### ثبّت LaTeX

يُستخدم مُترجمان (compilers) — `lualatex` للسيرة الذاتية، و`xelatex` لخطاب
التغطية. تتضمّن عمليات التثبيت أدناه كليهما.

<Tabs items={["macOS", "Linux", "Windows"]}>

<Tab value="macOS">

```bash title="Terminal"
brew install --cask mactex
sudo tlmgr install fontawesome lato tcolorbox dashrule enumitem multirow \
                   ifmtarg fontaxes mweights pgfopts
```

</Tab>

<Tab value="Linux">

```bash title="Terminal"
sudo apt-get install texlive-full
```

</Tab>

<Tab value="Windows">

ثبّت [MiKTeX](https://miktex.org/download) — فهو يجلب أي حزم ناقصة تلقائيًا
في أول مرة تُترجم فيها.

</Tab>

</Tabs>

**ما يحدث بعد ذلك:** لا شيء مرئي — يبقى LaTeX جاهزًا فحسب إلى أن يحتاج `/apply`
إلى إنتاج ملف PDF.

</Step>

<Step>

### احصل على CareerForge وافتحه

```bash title="Terminal"
git clone https://github.com/suraj-davariya/ai-job-search.git
cd ai-job-search
claude
```

**ما يحدث بعد ذلك:** يُفتح Claude Code *داخل* مجلد CareerForge، ما يعني أن
الأوامر الثلاثة أدناه أصبحت متاحة الآن في محادثتك.

</Step>

<Step>

### ابنِ ملفك الشخصي

```text title="In the Claude Code chat"
/setup
```

يعرض عليك ثلاثة مسارات: مسح مجلد من مستنداتك، أو استيراد سيرة ذاتية واحدة، أو
مقابلة ودّية يسأل فيها وتجيب أنت. اختر ما يناسب ما لديك. وقبل حفظ أي شيء، يعرض
عليك المحتوى المقترح ويطلب موافقتك.

**ما يحدث بعد ذلك:** تصبح ملفات ملفك الشخصي موجودة — خبرتك ومهاراتك وأهدافك
وتفضيلات بحثك، كلها على جهازك. تفعل هذا مرة واحدة فقط.

</Step>

<Step>

### اعثر على وظائف

```text title="In the Claude Code chat"
/search
```

**ما يحدث بعد ذلك:** جدول مُصنَّف بإعلانات جديدة من لوحات الوظائف التي ضبطتها،
كلٌّ منها معلَّم بملاءمة عالية أو متوسطة أو منخفضة — مع تصفية الإعلانات التي
رأيتها من قبل. ردّ برقم لتقييم أحدها بعمق.

</Step>

<Step>

### تقدّم إلى واحدة

```text title="In the Claude Code chat"
/apply https://company.com/jobs/the-role-you-want
```

يمكنك أيضًا لصق وصف الوظيفة مباشرةً إن لم يكن هناك رابط. يقيّم الملاءمة عبر
خمسة أبعاد ويسأل عمّا إذا كان ينبغي المتابعة — تلك نقطة تحقّقك، لا مجرد إجراء
شكلي.

**ما يحدث بعد ذلك:** سيرة ذاتية مخصّصة من صفحتين وخطاب تغطية من صفحة واحدة،
كلاهما بصيغة PDF، تمت مراجعتهما بواسطة ذكاء اصطناعي ثانٍ وفحصهما مقابل قائمة
تحقّق. يُسجَّل الطلب في متعقّبك بحالة **Draft**. تقرأ كلا ملفي PDF، ثم ترسلهما
بنفسك — لا يقدّم CareerForge أي شيء نيابةً عنك أبدًا.

</Step>

</Steps>

## بعد أول طلب تقديم لك

- افتح [لوحة المعلومات](/ar/docs/dashboard) لمتابعة مسار طلباتك وهو ينمو.
- اقرأ [بياناتك](/ar/docs/your-data) لترى بالضبط ما الذي سُجّل — إنه ملف واحد
  قابل للقراءة من البشر.
- متعثّر في أمرٍ ما؟ تغطّي [الأسئلة الشائعة](/ar/docs/faq) أسئلة الأسبوع الأول
  الشائعة.
