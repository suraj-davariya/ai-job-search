求職活動は、あなたが持つ最も個人的なデータを生み出します。どこで働いてきたか、
いくら稼いでいるか、どこへ移る意思があるか、どの企業に断られたか。CareerForge の
立場はシンプルです ― **そのデータはあなた以外の誰にも関係ない**ので、決して
あなたのコンピュータを離れません。

## あなたのデータがある場所

すべてがローカルに。

| データ | 保存場所 | アップロードされる？ |
|------|----------|----------------|
| 候補者プロフィール | あなたの CareerForge フォルダ | いいえ |
| 生成された CV とカバーレター | `cv/`、`cover_letters/` | いいえ |
| 応募トラッカー | `job_search_tracker.csv` | いいえ |
| 閲覧済み求人レジストリ | `job_scraper/` | いいえ |
| ソースドキュメント（CV、卒業証書…） | `documents/` | いいえ |
| 給与データ（任意） | `salary_data.json` | いいえ |

アカウントも、サインアップも、テレメトリも、分析も、「匿名の利用統計」も
ありません。このドキュメントサイトでさえ、何も取得しないように作られています ―
フォントやアセットはバンドルされており、サードパーティのサーバーから読み込まれません。

## ネットワークを通るもの ― そしてそのタイミング

正直であるためには全体像が必要です。2 つのものがインターネットに触れますが、
どちらもあなたが求めたときだけです。

1. **AI 自体。** CareerForge は Claude Code 上で動作するため、あなたの会話のテキスト ―
   AI が扱うプロフィールの内容を含む ― は [Anthropic のプライバシー条項](https://www.anthropic.com/privacy)の
   もとでそのサービスによって処理されます。これは他の用途で Claude を使うのと同じ
   取り決めです。あなたのチャットにとって許容できるなら、ここでも同じ仕組みです。
2. **あなたがトリガーするウェブ検索。** `/search` は公開されている求人サイトを
   クエリし、`/apply` は募集を取得して会社に関する主張を検証します。これらは
   通常のウェブ閲覧のように見えます。

決して起こらないこと: バックグラウンドのアップロード、あなたのファイルがファイルと
してどこかに送信されること、あるいは CareerForge のサーバー ― そんなものは存在しません。

## Git と、重要なたった 1 つの習慣

CareerForge には、上記のすべての個人的な成果物を除外する `.gitignore` が同梱されて
いるため、フォルダの履歴はデフォルトであなたのデータから清潔に保たれます。

守るべき習慣を一言で: **プライベートフォーク。** CareerForge のコピーを GitHub に
プッシュする場合は、リポジトリをプライベートにしてください。あなたのプロフィールは
コードと同じフォルダツリーにあり、無視ルールが守ってくれますが、プライベートな
リポジトリなら、ミスがあってもそれはあなたのものに留まります。

## すべてを削除する

閉じるアカウントも、サポートチケットもありません。フォルダを削除すれば、
CareerForge ― あなたの求職活動のあらゆる痕跡とともに ― は消えます。それが
ローカルファーストの意味です。
