Diese Seite bringt dich von einem leeren Rechner zu deinem ersten
maßgeschneiderten Lebenslauf und Anschreiben. Plane beim ersten Mal etwa
dreißig Minuten ein; das meiste davon ist die LaTeX-Installation, die im
Hintergrund still ihre Arbeit verrichtet.

## Was du zuerst brauchst

Sei ehrlich zu dir selbst bei diesen drei Voraussetzungen — jede verlinkt auf
eine ordentliche Installationsanleitung, und keine davon erfordert
Programmieren:

1. **Claude Code** — der KI-Assistent, in dem CareerForge läuft. Es ist ein
   Programm, das du einmal installierst und aus einem Terminalfenster
   öffnest. Noch nie ein Terminal benutzt? Kein Problem: Du tippst nur die
   kurzen Befehle ein, die auf dieser Seite gezeigt werden.
2. **LaTeX** — das Schriftsatzsystem, das deinen Lebenslauf in ein
   ausgefeiltes PDF verwandelt. Es ist ein großer Download, aber
   installieren-und-vergessen.
3. **Python 3.10+** *(optional)* — wird erst später für das
   Gehalts-Benchmarking benötigt. Überspring es heute.

<Steps>

<Step>

### Claude Code installieren

```bash title="Terminal"
npm install -g @anthropic-ai/claude-code
```

Dann einmal starten, um dich anzumelden:

```bash title="Terminal"
claude
```

Du brauchst ein [Anthropic-Konto](https://claude.ai) — sowohl der Free- als
auch der Pro-Plan funktionieren.

**Was als Nächstes passiert:** Ein Chat öffnet sich in deinem Terminal. Dieses
Chatfenster ist der Ort, an dem du alles Folgende ausführst.

</Step>

<Step>

### LaTeX installieren

Es werden zwei Compiler verwendet — `lualatex` für den Lebenslauf, `xelatex`
für das Anschreiben. Die unten stehenden Installationen enthalten beide.

<Tabs items={["macOS", "Linux", "Windows"]}>

<Tab value="macOS">

```bash title="Terminal"
brew install --cask mactex
sudo tlmgr install fontawesome lato tcolorbox dashrule enumitem multirow \
                   ifmtarg fontaxes mweights pgfopts
```

</Tab>

<Tab value="Linux">

```bash title="Terminal"
sudo apt-get install texlive-full
```

</Tab>

<Tab value="Windows">

Installiere [MiKTeX](https://miktex.org/download) — es holt fehlende Pakete
beim ersten Kompilieren automatisch nach.

</Tab>

</Tabs>

**Was als Nächstes passiert:** nichts Sichtbares — LaTeX wartet einfach bereit,
bis `/apply` ein PDF erzeugen muss.

</Step>

<Step>

### CareerForge holen und öffnen

```bash title="Terminal"
git clone https://github.com/suraj-davariya/ai-job-search.git
cd ai-job-search
claude
```

**Was als Nächstes passiert:** Claude Code öffnet sich *innerhalb* des
CareerForge-Ordners, was bedeutet, dass die drei unten stehenden Befehle nun in
deinem Chat verfügbar sind.

</Step>

<Step>

### Dein Profil aufbauen

```text title="Im Claude-Code-Chat"
/setup
```

Es bietet drei Wege an: einen Ordner mit deinen Dokumenten scannen, einen
einzelnen Lebenslauf importieren oder ein freundliches Interview, bei dem es
fragt und du antwortest. Wähle, was zu dem passt, was du hast. Bevor irgendetwas
gespeichert wird, zeigt es dir den vorgeschlagenen Inhalt und bittet um deine
Zustimmung.

**Was als Nächstes passiert:** Deine Profildateien existieren — deine
Erfahrung, Fähigkeiten, Ziele und Suchpräferenzen, alles auf deinem Rechner.
Das machst du nur einmal.

</Step>

<Step>

### Jobs finden

```text title="Im Claude-Code-Chat"
/search
```

**Was als Nächstes passiert:** eine bewertete Tabelle neuer Anzeigen aus deinen
konfigurierten Jobbörsen, jede als Hoch, Mittel oder Niedrig passend markiert —
wobei bereits gesehene Anzeigen herausgefiltert sind. Antworte mit einer Zahl,
um eine im Detail zu bewerten.

</Step>

<Step>

### Auf eine bewerben

```text title="Im Claude-Code-Chat"
/apply https://company.com/jobs/the-role-you-want
```

Du kannst die Stellenbeschreibung auch direkt einfügen, wenn es keinen Link
gibt. Es bewertet die Passung über fünf Dimensionen und fragt, ob fortgefahren
werden soll — das ist dein Kontrollpunkt, keine Formalität.

**Was als Nächstes passiert:** ein maßgeschneiderter zweiseitiger Lebenslauf und
ein einseitiges Anschreiben, beide als PDFs, von einer zweiten KI geprüft und
gegen eine Verifizierungsliste abgeglichen. Die Bewerbung wird in deinem Tracker
mit dem Status **Draft** festgehalten. Du liest beide PDFs und versendest sie
dann selbst — CareerForge reicht niemals etwas für dich ein.

</Step>

</Steps>
