यह पेज आपको एक ख़ाली मशीन से आपके पहले अनुकूलित CV और कवर लेटर तक ले जाता है। पहली
बार में लगभग तीस मिनट का अंदाज़ा रखें; इसका अधिकांश हिस्सा LaTeX इंस्टॉल चुपचाप
अपना काम करता रहता है।

## सबसे पहले आपको क्या चाहिए होगा

इन तीन पूर्व-आवश्यकताओं के बारे में अपने आप से ईमानदार रहें — हर एक एक उचित इंस्टॉल
गाइड से जुड़ी है, और इनमें से किसी के लिए प्रोग्रामिंग की ज़रूरत नहीं है:

1. **Claude Code** — वह AI सहायक जिसके अंदर CareerForge चलता है। यह एक प्रोग्राम है
   जिसे आप एक बार इंस्टॉल करते हैं और एक टर्मिनल विंडो से खोलते हैं। कभी टर्मिनल का
   इस्तेमाल नहीं किया? कोई बात नहीं: आप सिर्फ़ इस पेज पर दिखाए गए छोटे कमांड ही टाइप
   करेंगे।
2. **LaTeX** — वह टाइपसेटिंग सिस्टम जो आपके CV को एक परिष्कृत PDF में बदल देता है।
   यह एक बड़ा डाउनलोड है, लेकिन इंस्टॉल-करो-और-भूल जाओ वाला है।
3. **Python 3.10+** *(वैकल्पिक)* — केवल बाद में सैलरी बेंचमार्किंग के लिए ज़रूरी।
   आज इसे छोड़ दें।

<Steps>

<Step>

### Claude Code इंस्टॉल करें

```bash title="Terminal"
npm install -g @anthropic-ai/claude-code
```

फिर लॉग इन करने के लिए इसे एक बार शुरू करें:

```bash title="Terminal"
claude
```

आपको एक [Anthropic खाते](https://claude.ai) की ज़रूरत होगी — Free और Pro दोनों
प्लान काम करते हैं।

**आगे क्या होता है:** आपके टर्मिनल में एक चैट खुलती है। वही चैट विंडो वह जगह है
जहाँ आप नीचे दिया गया सब कुछ चलाएँगे।

</Step>

<Step>

### LaTeX इंस्टॉल करें

दो कम्पाइलर इस्तेमाल होते हैं — CV के लिए `lualatex`, कवर लेटर के लिए `xelatex`।
नीचे दिए गए इंस्टॉल दोनों को शामिल करते हैं।

<Tabs items={["macOS", "Linux", "Windows"]}>

<Tab value="macOS">

```bash title="Terminal"
brew install --cask mactex
sudo tlmgr install fontawesome lato tcolorbox dashrule enumitem multirow \
                   ifmtarg fontaxes mweights pgfopts
```

</Tab>

<Tab value="Linux">

```bash title="Terminal"
sudo apt-get install texlive-full
```

</Tab>

<Tab value="Windows">

[MiKTeX](https://miktex.org/download) इंस्टॉल करें — पहली बार कम्पाइल करते समय यह
किसी भी छूटे हुए पैकेज को अपने आप ले आता है।

</Tab>

</Tabs>

**आगे क्या होता है:** कुछ भी दिखाई देने वाला नहीं — LaTeX बस तब तक तैयार बैठा रहता
है जब तक `/apply` को एक PDF बनाने की ज़रूरत न पड़े।

</Step>

<Step>

### CareerForge पाएँ और उसे खोलें

```bash title="Terminal"
git clone https://github.com/suraj-davariya/ai-job-search.git
cd ai-job-search
claude
```

**आगे क्या होता है:** Claude Code CareerForge फ़ोल्डर के *अंदर* खुलता है, जिसका
मतलब है कि नीचे दिए गए तीन कमांड अब आपकी चैट में उपलब्ध हैं।

</Step>

<Step>

### अपनी प्रोफ़ाइल बनाएँ

```text title="In the Claude Code chat"
/setup
```

यह तीन रास्ते देता है: अपने दस्तावेज़ों के एक फ़ोल्डर को स्कैन करना, एक अकेला CV
इम्पोर्ट करना, या एक दोस्ताना इंटरव्यू जहाँ यह पूछता है और आप जवाब देते हैं। जो
आपके पास है उसके हिसाब से कोई भी चुनें। कुछ भी सहेजे जाने से पहले, यह आपको
प्रस्तावित सामग्री दिखाता है और आपकी मंज़ूरी माँगता है।

**आगे क्या होता है:** आपकी प्रोफ़ाइल फ़ाइलें मौजूद हो जाती हैं — आपका अनुभव, कौशल,
लक्ष्य, और खोज प्राथमिकताएँ, सब आपकी मशीन पर। आप यह सिर्फ़ एक बार करते हैं।

</Step>

<Step>

### नौकरियाँ ढूँढें

```text title="In the Claude Code chat"
/search
```

**आगे क्या होता है:** आपके कॉन्फ़िगर किए गए जॉब बोर्डों से नई पोस्टिंग की एक रैंक की
हुई तालिका, हर एक पर High, Medium, या Low फ़िट का निशान — और जो पोस्टिंग आप पहले देख
चुके हैं वे छाँट दी जाती हैं। किसी एक का गहराई से मूल्यांकन करने के लिए एक नंबर के
साथ जवाब दें।

</Step>

<Step>

### किसी एक के लिए आवेदन करें

```text title="In the Claude Code chat"
/apply https://company.com/jobs/the-role-you-want
```

अगर कोई लिंक न हो तो आप जॉब डिस्क्रिप्शन सीधे पेस्ट भी कर सकते हैं। यह पाँच आयामों
में फ़िट का स्कोर करता है और पूछता है कि आगे बढ़ें या नहीं — यह आपका चेकपॉइंट है, कोई
औपचारिकता नहीं।

**आगे क्या होता है:** एक अनुकूलित दो-पन्नों का CV और एक एक-पन्ने का कवर लेटर, दोनों
PDF के रूप में, एक दूसरे AI द्वारा समीक्षित और एक सत्यापन सूची के विरुद्ध जाँचे हुए।
आवेदन आपके ट्रैकर में **Draft** स्थिति के साथ दर्ज हो जाता है। आप दोनों PDF पढ़ते
हैं, फिर उन्हें ख़ुद भेजते हैं — CareerForge आपके लिए कभी कुछ जमा नहीं करता।

</Step>

</Steps>
