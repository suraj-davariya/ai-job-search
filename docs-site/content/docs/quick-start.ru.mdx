Эта страница проведёт вас от пустой машины до вашего первого резюме и
сопроводительного письма под вакансию. В первый раз заложите около тридцати
минут; большую часть из них тихо занимает установка LaTeX.

## Что понадобится сначала

Будьте честны с собой насчёт этих трёх предварительных условий — каждое ведёт
к нормальному руководству по установке, и ни одно из них не требует
программирования:

1. **Claude Code** — ИИ-ассистент, внутри которого работает CareerForge. Это
   программа, которую вы устанавливаете один раз и открываете из окна терминала.
   Никогда не пользовались терминалом? Ничего страшного: вы будете вводить
   только короткие команды, показанные на этой странице.
2. **LaTeX** — система вёрстки, которая превращает ваше резюме в отполированный
   PDF. Загрузка большая, но «поставил и забыл».
3. **Python 3.10+** *(необязательно)* — понадобится только позже для сравнения
   зарплат. Сегодня пропустите.

<Steps>

<Step>

### Установите Claude Code

```bash title="Terminal"
npm install -g @anthropic-ai/claude-code
```

Затем запустите его один раз для входа:

```bash title="Terminal"
claude
```

Вам понадобится [учётная запись Anthropic](https://claude.ai) — подойдут как
тариф Free, так и Pro.

**Что произойдёт дальше:** в терминале откроется чат. Это окно чата — то место,
где вы будете выполнять всё, что описано ниже.

</Step>

<Step>

### Установите LaTeX

Используются два компилятора — `lualatex` для резюме и `xelatex` для
сопроводительного письма. Установки ниже включают оба.

<Tabs items={["macOS", "Linux", "Windows"]}>

<Tab value="macOS">

```bash title="Terminal"
brew install --cask mactex
sudo tlmgr install fontawesome lato tcolorbox dashrule enumitem multirow \
                   ifmtarg fontaxes mweights pgfopts
```

</Tab>

<Tab value="Linux">

```bash title="Terminal"
sudo apt-get install texlive-full
```

</Tab>

<Tab value="Windows">

Установите [MiKTeX](https://miktex.org/download) — он автоматически дозагружает
любые недостающие пакеты при первой компиляции.

</Tab>

</Tabs>

**Что произойдёт дальше:** ничего видимого — LaTeX просто стоит наготове, пока
`/apply` не понадобится сформировать PDF.

</Step>

<Step>

### Получите CareerForge и откройте его

```bash title="Terminal"
git clone https://github.com/suraj-davariya/ai-job-search.git
cd ai-job-search
claude
```

**Что произойдёт дальше:** Claude Code открывается *внутри* папки CareerForge,
а значит три команды ниже теперь доступны в вашем чате.

</Step>

<Step>

### Постройте свой профиль

```text title="In the Claude Code chat"
/setup
```

Он предлагает три пути: сканировать папку с вашими документами, импортировать
одно резюме или дружеское интервью, где он спрашивает, а вы отвечаете. Выберите
то, что подходит под имеющееся у вас. Прежде чем что-либо сохранить, он
показывает вам предлагаемое содержимое и просит вашего одобрения.

**Что произойдёт дальше:** файлы вашего профиля созданы — ваш опыт, навыки, цели
и предпочтения по поиску, всё на вашей машине. Это делается только один раз.

</Step>

<Step>

### Найдите вакансии

```text title="In the Claude Code chat"
/search
```

**Что произойдёт дальше:** ранжированная таблица новых объявлений с настроенных
вами сайтов вакансий, каждое помечено как Высокое, Среднее или Низкое
соответствие — с отфильтрованными объявлениями, которые вы уже видели. Ответьте
номером, чтобы оценить одно из них подробно.

</Step>

<Step>

### Подайтесь на одну

```text title="In the Claude Code chat"
/apply https://company.com/jobs/the-role-you-want
```

Вы также можете вставить описание вакансии напрямую, если ссылки нет. Он
оценивает соответствие по пяти измерениям и спрашивает, продолжать ли — это ваша
контрольная точка, а не формальность.

**Что произойдёт дальше:** двухстраничное резюме под вакансию и одностраничное
сопроводительное письмо, оба в виде PDF, проверенные вторым ИИ и сверенные со
списком верификации. Заявка зафиксирована в вашем трекере со статусом
**Draft**. Вы читаете оба PDF, а затем отправляете их сами — CareerForge никогда
ничего не отправляет за вас.

</Step>

</Steps>

## После вашей первой заявки

- Откройте [панель управления](/ru/docs/dashboard), чтобы наблюдать за ростом
  вашей воронки.
- Прочитайте [Ваши данные](/ru/docs/your-data), чтобы увидеть, что именно
  записалось, — это один читаемый человеком файл.
- Застряли на чём-то? [FAQ](/ru/docs/faq) охватывает частые вопросы первой
  недели.
