`/reset` هو زر البداية النظيفة. أحيانًا تريد أن تبدأ من جديد — اتجاه مهني جديد، أو
ملف تعريفي جديد، أو أنك تسلّم المستودع لشخص آخر. يزيل `/reset` **بياناتك** مع الحفاظ
بعناية على **إطار العمل** الذي يجعل CareerForge يعمل، بحيث تستمر الأداة في العمل بشكل
مثالي بعد ذلك.

## اختر ما تريد مسحه

| ما تكتبه | يمسح |
|----------|--------|
| `/reset profile` | ملفات مهارة ملفك التعريفي — تعود إلى قوالب فارغة |
| `/reset documents` | ملفاتك في `documents/` (المجلدات وملف README تبقى) |
| `/reset all` | كلاهما |

إذا لم تمرّر نطاقًا، فإنه يطلب منك الاختيار ويشرح ما يؤثر عليه كل خيار.

## يعرض لك كل شيء أولاً

قبل حذف أي شيء، يطبع `/reset` **جردًا**:

```text
you:    /reset profile
claude: Profile reset — here's what I'll clear vs. keep:

        Will clear:
          01-candidate-profile.md     (has your data)
          02-behavioral-profile.md    (has your data)
          07 STAR examples            (3 examples)
        Already empty:
          05 profile statements       (template)
        NEVER touched (framework):
          03-writing-style.md, 04 scoring framework,
          06 cover-letter templates, 07 framework + roleplay

        This cannot be undone — your git history is the only recovery.
        Type RESET to proceed.
you:    RESET
claude: Cleared 3 files. Run /setup to rebuild your profile.
```

## حواجز الأمان

- **الجرد قبل الحذف** — ترى دائمًا بالضبط ما الذي سيُحذف وما الذي سيبقى.
- **يجب أن تكتب `RESET`** بحروف كبيرة. أي رد آخر يلغي العملية مع رسالة "لم يتغير
  شيء".
- **إطار العمل لا يُمسّ أبدًا** — دليل أسلوب الكتابة، وإطار التقييم، وقوالب خطاب
  التغطية، وإطار التحضير للمقابلات (بما في ذلك إرشادات لعب الأدوار الخاصة بك) كلها
  تبقى. تُمسح بياناتك *فقط*.
- **إبلاغ صادق** — يسرد ما تم مسحه فعليًا مقابل ما كان فارغًا بالفعل، ولا يدّعي
  أبدًا أنه مسح شيئًا لم يكن موجودًا.
- **لا تراجع.** يقول ذلك مسبقًا ويوجّهك إلى سجل git الخاص بك.

## بعد إعادة الضبط

يخبرك `/reset` بالخطوة التالية الطبيعية لما مسحته — وهي دائمًا تقريبًا
[`/setup`](/ar/docs/commands/setup) لإعادة بناء ملفك التعريفي (أضف المستندات أولاً إذا
مسحتها).

> لأنه مدمّر، فإن `/reset` غير متاح عمدًا من وحدة تحكم لوحة المعلومات
> Console — فهو يعمل هنا فقط، حيث يحميك تأكيد `RESET` المكتوب.
