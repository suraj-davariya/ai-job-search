এই পৃষ্ঠাটি আপনাকে একটি ফাঁকা মেশিন থেকে আপনার প্রথম উপযোগী CV ও কভার লেটার পর্যন্ত
নিয়ে যাবে। প্রথমবার প্রায় ত্রিশ মিনিটের পরিকল্পনা করুন; এর বেশিরভাগটাই LaTeX
ইনস্টল নীরবে নিজের কাজ করার সময়।

## প্রথমে যা যা লাগবে

এই তিনটি পূর্বশর্ত নিয়ে নিজের সাথে সৎ থাকুন — প্রতিটির সাথে একটি যথাযথ ইনস্টল
গাইডের লিঙ্ক আছে, এবং কোনোটিতেই প্রোগ্রামিং লাগে না:

1. **Claude Code** — যে AI সহকারীর ভেতরে CareerForge চলে। এটি একটি প্রোগ্রাম যা
   একবার ইনস্টল করেন এবং একটি টার্মিনাল উইন্ডো থেকে খোলেন। কখনো টার্মিনাল ব্যবহার
   করেননি? কোনো সমস্যা নেই: আপনি কেবল এই পৃষ্ঠায় দেখানো ছোট কমান্ডগুলো টাইপ করবেন।
2. **LaTeX** — যে টাইপসেটিং সিস্টেম আপনার CV-কে একটি পরিপাটি PDF-এ পরিণত করে।
   এটি একটি বড় ডাউনলোড, কিন্তু একবার ইনস্টল করলে আর ভাবতে হয় না।
3. **Python 3.10+** *(ঐচ্ছিক)* — পরে কেবল বেতন বেঞ্চমার্কিংয়ের জন্য দরকার।
   আজ এটি বাদ দিন।

<Steps>

<Step>

### Claude Code ইনস্টল করুন

```bash title="Terminal"
npm install -g @anthropic-ai/claude-code
```

তারপর লগ ইন করতে একবার এটি চালু করুন:

```bash title="Terminal"
claude
```

আপনার একটি [Anthropic account](https://claude.ai) লাগবে — Free এবং Pro উভয় প্ল্যানই
কাজ করে।

**এরপর যা ঘটে:** আপনার টার্মিনালে একটি চ্যাট খোলে। সেই চ্যাট উইন্ডোতেই আপনি নিচের
সবকিছু চালাবেন।

</Step>

<Step>

### LaTeX ইনস্টল করুন

দুটি কম্পাইলার ব্যবহার করা হয় — CV-এর জন্য `lualatex`, কভার লেটারের জন্য
`xelatex`। নিচের ইনস্টলগুলোতে উভয়ই অন্তর্ভুক্ত।

<Tabs items={["macOS", "Linux", "Windows"]}>

<Tab value="macOS">

```bash title="Terminal"
brew install --cask mactex
sudo tlmgr install fontawesome lato tcolorbox dashrule enumitem multirow \
                   ifmtarg fontaxes mweights pgfopts
```

</Tab>

<Tab value="Linux">

```bash title="Terminal"
sudo apt-get install texlive-full
```

</Tab>

<Tab value="Windows">

[MiKTeX](https://miktex.org/download) ইনস্টল করুন — প্রথমবার কম্পাইল করার সময় এটি
যেকোনো অনুপস্থিত প্যাকেজ স্বয়ংক্রিয়ভাবে নামিয়ে নেয়।

</Tab>

</Tabs>

**এরপর যা ঘটে:** দৃশ্যমান কিছুই না — `/apply`-এর কোনো PDF তৈরি করার দরকার না পড়া
পর্যন্ত LaTeX কেবল প্রস্তুত হয়ে বসে থাকে।

</Step>

<Step>

### CareerForge নিয়ে আসুন এবং খুলুন

```bash title="Terminal"
git clone https://github.com/suraj-davariya/ai-job-search.git
cd ai-job-search
claude
```

**এরপর যা ঘটে:** Claude Code CareerForge ফোল্ডারের *ভেতরে* খোলে, যার মানে নিচের
তিনটি কমান্ড এখন আপনার চ্যাটে উপলব্ধ।

</Step>

<Step>

### আপনার প্রোফাইল তৈরি করুন

```text title="In the Claude Code chat"
/setup
```

এটি তিনটি পথ দেয়: আপনার নথিপত্রের একটি ফোল্ডার স্ক্যান করা, একটি একক CV ইম্পোর্ট
করা, কিংবা একটি বন্ধুত্বপূর্ণ ইন্টারভিউ যেখানে এটি জিজ্ঞেস করে আর আপনি উত্তর দেন।
আপনার কাছে যা আছে তার সাথে মানানসই যেটি, সেটি বেছে নিন। কিছু সংরক্ষণ করার আগে এটি
আপনাকে প্রস্তাবিত বিষয়বস্তু দেখায় এবং আপনার অনুমোদন চায়।

**এরপর যা ঘটে:** আপনার প্রোফাইল ফাইলগুলো তৈরি হয় — আপনার অভিজ্ঞতা, দক্ষতা, লক্ষ্য
এবং খোঁজার পছন্দ, সবই আপনার মেশিনে। এটি আপনি একবারই করেন।

</Step>

<Step>

### চাকরি খুঁজুন

```text title="In the Claude Code chat"
/search
```

**এরপর যা ঘটে:** আপনার কনফিগার করা জব বোর্ডগুলো থেকে নতুন পোস্টিংয়ের একটি র‍্যাঙ্ক
করা টেবিল, প্রতিটি High, Medium বা Low ফিট হিসেবে চিহ্নিত — আগে দেখা পোস্টিংগুলো
ছেঁকে বাদ দেওয়া। গভীরভাবে কোনোটি মূল্যায়ন করতে একটি নম্বর দিয়ে উত্তর দিন।

</Step>

<Step>

### একটিতে আবেদন করুন

```text title="In the Claude Code chat"
/apply https://company.com/jobs/the-role-you-want
```

কোনো লিঙ্ক না থাকলে আপনি সরাসরি জব ডেসক্রিপশনটিও পেস্ট করতে পারেন। এটি পাঁচটি
মাত্রা জুড়ে ফিট স্কোর করে এবং এগোবে কিনা জিজ্ঞেস করে — এটি আপনার চেকপয়েন্ট, কোনো
আনুষ্ঠানিকতা নয়।

**এরপর যা ঘটে:** একটি উপযোগী দুই-পাতার CV এবং একটি এক-পাতার কভার লেটার, দুটিই
PDF হিসেবে, একটি দ্বিতীয় AI দ্বারা পর্যালোচিত এবং একটি যাচাই তালিকার বিপরীতে
পরীক্ষিত। আবেদনটি আপনার ট্র্যাকারে **Draft** স্ট্যাটাসসহ লগ হয়।
আপনি দুটি PDF পড়েন, তারপর নিজে সেগুলো পাঠান — CareerForge কখনো আপনার হয়ে কিছু
জমা দেয় না।

</Step>

</Steps>

## আপনার প্রথম আবেদনের পর

- আপনার পাইপলাইন বড় হওয়ার সাথে সাথে তা দেখতে [ড্যাশবোর্ড](/bn/docs/dashboard) খুলুন।
- ঠিক কী রেকর্ড হলো তা দেখতে [Your data](/bn/docs/your-data) পড়ুন — এটি একটি
  মানুষ-পাঠযোগ্য ফাইল।
- কোনো কিছুতে আটকে গেছেন? [FAQ](/bn/docs/faq)-তে সাধারণ প্রথম-সপ্তাহের প্রশ্নগুলোর
  উত্তর আছে।
